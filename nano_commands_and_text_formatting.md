
# Nano Commands

- **Save File**: `Ctrl + O`
- **Exit nano**: `Ctrl + X`
- **Cut Text**: `Ctrl + K`
- **Paste Text**: `Ctrl + U`
- **Search**: `Ctrl + W`
- **Replace**: `Ctrl + \`
- **Mark Text**: `Ctrl + ^` (Ctrl + Shift + 6)
- **Justify Text**: `Ctrl + J`

---

# Text Formatting

- Italic: 
  - `*This is italic text*`
  - `_this is also italic_`
  
- Bold: 
  - `**This is bold text**`
  - `__this is also bold__`
